## ūüíĽ What is Uplink3?

**Uplink3** is a Python terminal game inspired by the classic **Uplink** hacker simulation game. You play as a hacker navigating a command-line world ‚ÄĒ cracking systems, completing missions, and staying one step ahead. All inside your terminal.

No GUI. No mouse. Just you, a keyboard, and a blinking cursor.

---

## ūüéģ About the Original Uplink

**Uplink** (2001) by Introversion Software is a legendary hacker simulation game where you act as a freelance hacker, taking on contracts to infiltrate computer systems, steal data, and cover your tracks. Uplink3 brings that same dark terminal atmosphere to Python.

## ūüĒß Troubleshooting

| Problem | Fix |
|---|---|
| `ModuleNotFoundError: No module named 'colorama'` | Run `pip install colorama` |
| `python: command not found` | Try `python3 uplink3.py` instead |
| `apt: command not found` | You're not on Debian/Ubuntu/Termux ‚ÄĒ use your distro's package manager |
| Colors not showing on Windows | Make sure you're using **Windows Terminal** or **PowerShell**, not the old CMD |
| Game crashes on startup | Make sure Python 3.x is installed, not Python 2 |
